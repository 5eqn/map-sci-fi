# 映射

计划写成一个关于依值类型、函数式编程、人工智能、量子芯片和脑机接口的不软不硬的科幻。

不软在于我希望不引入太过离谱的特性，希望整体给人一种未来真的可能实现的感觉。

不硬在于我计科功底不扎实，没有能力呈现较为严谨的内容，因此也可能有很多设想一眼不能实现。
